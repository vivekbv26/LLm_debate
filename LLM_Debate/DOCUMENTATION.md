## Examples Directory

The `examples/` directory contains 12 working examples:

| Example | Description | Command |
|---------|-------------|---------|
| `basic-debate.js` | Simple two-agent debate | `npm run example` |
| `math-problem.js` | Mathematical problem solving | `npm run example:math` |
| `coding-task.js` | Coding with multiple agents | `npm run example:coding` |
| `research-task.js` | Research with RAG | `npm run example:research` |
| `orchestrator-mode.js` | Intelligent routing | `npm run example:orchestrator` |
| `custom-agents.js` | Custom agent examples | `npm run example:custom` |
| `multiple-models.js` | Mix different LLM providers | `npm run example:models` |
| `user-defined-custom.js` | User-created specialists | `npm run example:user` |
| `simple-custom.js` | Simplest custom agent | - |
| `common-response-evolution.js` | See response evolve | `npm run example:evolution` |
| `advanced-all-agents.js` | All built-in agents | - |
| `ultimate-showcase.js` | Everything together | `npm run example:ultimate` |
